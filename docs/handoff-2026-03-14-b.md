### Architecture Summary

**Before:** Upload → pre-scale image to intermediate canvas → `toDataURL()` → create Image from data URL → tile the pre-scaled image. Pixelated because the intermediate canvas was small.

**After:** Upload → store original HTMLImageElement → on each render, calculate which tiles are visible in the viewport → draw each tile directly from the full-res source via `ctx.drawImage(img, sx, sy, sw, sh, dx, dy, dw, dh)`. Scale factor = `(zoom / 100) * (96 / dpi)`.
