## Architecture Notes

### Double Buffering Pattern
Both canvases now use this pattern:
```
offscreenRef (reused HTMLCanvasElement)
  ŌåÆ draw all tiles + outline to offscreen
  ŌåÆ ctx.drawImage(offscreen, 0, 0)  ŌåÉ single atomic blit to visible canvas
```
The offscreen canvas is stored in a ref and only resized when dimensions actually change.

### Touch Handling Pattern (non-passive)
Pinch zoom requires `e.preventDefault()` to block browser page-zoom, which doesn't work in React synthetic events (passive by default). Both canvases attach touch listeners via `useEffect` with `{ passive: false }` on `touchmove`.

### SeamInspector Floating Control
- `position: fixed` removes it from the stacking context entirely
- Drag math uses viewport coordinates (matches `getBoundingClientRect()` for fixed elements)
- On first drag, measures real position from DOM to switch from `bottom: 20` to absolute `top: Y`
