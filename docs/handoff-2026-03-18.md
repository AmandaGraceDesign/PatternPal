### Pattern Fill Export — Social Media Path

The Pattern Fill Export modal was redesigned to serve two audiences: Cricut/Silhouette (existing, unchanged) and social media creators (new).

**Entry point:** "Pattern Fill Export" button in AdvancedToolsBar → destination picker screen (new) → Cricut or Social path.

---

### Social Media Export — Wizard Flow

**Step 1 — Select sizes**
- Format toggle: JPG (default) / PNG
- Select All checkbox (with indeterminate state) + 5 individual size checkboxes:
  - Instagram / Facebook Post (1080×1080)
  - Instagram / Facebook Portrait (1080×1350)
  - Story / Reel / TikTok (1080×1920)
  - Pinterest Pin (1000×1500)
  - Facebook Cover (1640×624)
- "Preview & Export" button (disabled until ≥1 size selected)

**Step 2 — Per-size preview wizard**
- One size per slide with a large 240px canvas preview
- Independent −/+ scale controls per size (jumps to next meaningful repeat count)
- Shows: tile size in inches, repeat count (e.g. "6 × 10 repeats"), pixel dimensions, filename preview
- ← Back to Sizes / Next → navigation
- Last slide: Export button ("Export 1 Image" / "Export N Images")
- Scale state persists if you navigate back and forward between slides

**Export behavior**
- Single pro gate (`POST /api/pro/verify`) before rendering
- 1 size checked → direct download
- 2+ sizes checked → zip file (`{pattern}-social-media.zip`)
- Per-size failures are skipped gracefully; partial failure shows inline warning
