### Session 2 (this session)

#### 6. Mockup Overlay on Social Media Export (NEW FEATURE)
Users can overlay one of 6 product mockups on each social media export image with a white photo border and drop shadow.

**How it works:**
- **Per-slide controls** on Step 2 preview slides — "Add Mockup" checkbox + horizontal thumbnail picker
- **6 mockups:** Onesie, Fabric Swatch, Wallpaper, Throw Pillow, Wrapping Paper, Journal
- **Selection is per-size** — different mockups for different sizes, or none
- **Mockup renders** at ~55% of shorter canvas dimension, centered with 2.5% white border + drop shadow
- **Layering order:** Pattern fill → Mockup overlay → Watermark
- **Mobile-friendly:** 44px touch targets, flex-wrap thumbnails, modal horizontal margin on phones

**Architecture:**
- New `renderMockupOffscreen.ts` utility extracts MockupRenderer's full canvas compositing pipeline (masks, multiply blend, onesie/wrapping paper special cases) into a reusable async function
- `mockupsRef` (ref-based, like `scalesRef`) stores per-size mockup config without causing re-renders
- `drawMockupOverlay()` centers mockup with white border + shadow
- `applyMockupOverlay()` stamps mockup onto export blob

**Export pipeline:**
```
generateSocialFillBlob() → applyMockupOverlay() → applyWatermarkToBlob() → download
```

#### 7. Hairline Gap Fix — Main Canvas & Seam Inspector
**Problem:** Dark background colors (`#0f172a` in PatternTiler, `#294051` in SeamInspectorCanvas) bled through sub-pixel anti-aliasing gaps at tile boundaries, causing visible hairline dark lines.
**Fix:** Changed both canvas backgrounds to `#ffffff`. Micro-gaps are now invisible against white.
