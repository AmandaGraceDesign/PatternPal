## Why Pinterest for PatternPAL Pro

- **Your audience lives here.** Surface pattern designers use Pinterest daily for inspiration boards, color palettes, and trend research.
- **Visual-first = perfect product fit.** PatternPAL's output (tiled patterns, product mockups, social exports) is inherently pin-worthy content.
- **High-intent discovery.** Pinterest users are planners — they search with buying intent ("surface pattern design tools," "seamless repeat tutorial"). This is closer to search than social.
- **Long pin lifespan.** A pin can drive traffic for 3-6 months vs. 48 hours on Instagram/TikTok.
- **Lower CPCs.** Pattern design is a niche with low ad competition on Pinterest. Expect $0.30-$0.80 CPCs vs. $1-3 on Google Search.

## Tracking Setup — Pinterest Tag

### Tag Installation

Pinterest Tag ID: `[TO BE ASSIGNED]`

**Option A: GTM (Recommended — you already have GTM-5PM5T8RC)**
1. In GTM, add a new tag: **Pinterest Tag** (Community Template Gallery)
2. Enter your Pinterest Tag ID
3. Trigger: All Pages
4. Add event tags for conversions (see below)

**Option B: Direct installation in layout.tsx**
Add to `<head>` after the existing GTM/gtag scripts:

```html
<!-- Pinterest Tag -->
<script>
  !function(e){if(!window.pintrk){window.pintrk=function(){window.pintrk.queue.push(Array.prototype.slice.call(arguments))};var n=window.pintrk;n.queue=[],n.version="3.0";var t=document.createElement("script");t.async=!0,t.src=e;var r=document.getElementsByTagName("script")[0];r.parentNode.insertBefore(t,r)}}("https://s.pinimg.com/ct/core.js");
  pintrk('load', '[PINTEREST_TAG_ID]');
  pintrk('page');
</script>
```

### Conversion Events

| Event | Pinterest Event | Trigger | Value |
|-------|----------------|---------|-------|
| Page Visit | `pagevisit` | Every page load | — |
| Free Signup | `signup` | Clerk account created (<60s) | $0 |
| Pro Trial Monthly | `checkout` | Stripe redirect with `plan=monthly` | $7.99 |
| Pro Trial Annual | `checkout` | Stripe redirect with `plan=yearly` | $79.92 |

**GTM Implementation:**
1. Create a Custom HTML tag for each conversion event
2. Trigger `signup` on the same condition as the Google Ads SignupConversion (Clerk `isSignedIn` + `createdAt` < 60s)
3. Trigger `checkout` on the same condition as CheckoutConversion (URL contains `checkout=success`)

**Direct Implementation (mirror your Google Ads approach):**
Create `src/lib/pinterest.ts`:
```ts
export function firePinterestEvent(
  event: 'signup' | 'checkout',
  data?: { value?: number; currency?: string }
) {
  if (typeof window !== 'undefined' && window.pintrk) {
    window.pintrk('track', event, {
      value: data?.value,
      currency: data?.currency || 'USD',
    });
  }
}
```

Then call alongside your existing Google Ads conversions in `SignupConversion.tsx` and `CheckoutConversion.tsx`.

### CSP Updates Required

Add to `next.config.ts` Content Security Policy:
- `script-src`: `https://s.pinimg.com`
- `img-src`: `https://ct.pinterest.com`
- `connect-src`: `https://ct.pinterest.com`

### Enhanced Match

Enable Enhanced Match in Pinterest Ads Manager to pass hashed email for better attribution:
```js
pintrk('load', '[PINTEREST_TAG_ID]', { em: '<hashed_email>' });
```

You can pass the Clerk user email (hashed) when the user is signed in.

## Pinterest vs. Google Ads — Key Differences

| | Google Search | Pinterest |
|---|---|---|
| **Intent** | "I need this now" | "I'm exploring and planning" |
| **Creative** | Text ads (headlines + descriptions) | Visual pins (images/video required) |
| **Funnel position** | Bottom (direct response) | Mid-to-top (discovery + consideration) |
| **Conversion timeline** | Same session | 7-30 day consideration window |
| **Attribution** | Last click (mostly) | View-through + click (longer window) |
| **Lifespan** | Ad stops when budget stops | Pins continue driving organic traffic |
| **CPC range (est.)** | $1-3 for pattern keywords | $0.30-0.80 for pattern keywords |
| **Creative effort** | Low (text only) | Medium-high (images, video, idea pins) |

**Key takeaway:** Pinterest is a slower burn than Google Search. Expect conversions to take 1-3 weeks to ramp up. The attribution window is longer, so don't kill campaigns too early. Give each campaign at least 2-3 weeks of data before making major changes.

## Step-by-Step Setup Walkthrough

### 1. Create Business Account
1. Go to [business.pinterest.com](https://business.pinterest.com)
2. Create account or convert your existing Amanda Grace Design account
3. Complete business profile (name, website, description)

### 2. Claim Website
1. Go to **Settings > Claim**
2. Add `pattern-tester.amandagracedesign.com`
3. Choose HTML tag method (add meta tag to `<head>` in layout.tsx) or HTML file upload
4. Verify

### 3. Install Pinterest Tag
1. Go to **Ads > Conversions**
2. Create a new Pinterest Tag
3. Copy the Tag ID
4. Install via GTM or direct code (see Tracking section above)
5. Install **Pinterest Tag Helper** Chrome extension to verify

### 4. Set Up Conversion Events
1. In **Ads > Conversions**, create events:
   - `pagevisit` — automatic with base tag
   - `signup` — custom event on account creation
   - `checkout` — custom event on Stripe success redirect
2. Set conversion values ($0, $7.99, $79.92)

### 5. Create Campaigns
1. Go to **Ads > Create campaign**
2. Choose **Conversions** objective for Campaigns 1 & 2
3. Set daily budget, geo targeting, and placements
4. Create ad groups with keywords + interest targeting
5. Upload pin creative (see Pin Creative Strategy above)

### 6. Upload Creative
1. Create pins in **Ads > Pins** or upload during campaign creation
2. Use 1000x1500px (2:3 ratio) for standard pins
3. Add destination URL: `https://pattern-tester.amandagracedesign.com?utm_source=pinterest&utm_medium=cpc&utm_campaign=[campaign_name]`
4. Add UTM parameters for Google Analytics tracking

### 7. Launch & Monitor
1. Set campaigns live
2. Check Pinterest Tag Helper daily for first 3 days
3. Monitor **Ads Manager > Analytics** for impressions, clicks, CTR, CPA
4. Follow the Optimization Milestones above
